About This Website

News Website

This website allows users to view, vote on and comment on articles in a reddit style interface. It is built to be mobile responsive.

I built the front end of this website using React. The backend of the site is built using node.js and express which is complete with a full test suite using jest. The database is created using PSQL.

The website is fully hosted, click here to view it. Click here for the github repo.

Video Uploading App - NC Portfolio

As a group of five we built this app demo which will run on android or ios.

The idea for this app was to provide a place for software developers to display demo videos of their creations.

We self organised through out, having daily standups and creating a trello ticket board to organise our work. We reviewed each others code before merging it into the main branch on github.

The front end of this app is built using react-native, the backend is built using express and node.js and with integration testing using jest. We hosted our videos on cloudinary and built our database using PSQL.
